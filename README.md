# svelte-codemirror-search-conceal-poc

POC of an enriched search field using CodeMirror and the conceal mechanism in a Svelte application

## AI-Assisted Development

This project was developed using:

- [OpenCode](https://opencode.ai) CLI — coding assistant workflow (not vibe coding)
- Models: DeepSeek V4 Flash

## Prerequisite

Install [mise](https://mise.jdx.dev/getting-started.html) — it will handle installing Node.js and pnpm for you.

## Getting Started

```bash
$ mise install
$ pnpm install
```
