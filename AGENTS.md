# Agent Instructions

## Project Context

POC of an enriched search field using CodeMirror and the conceal mechanism in a Svelte application

## Language Policy

- **All project content must be in English**: source code, comments, commit messages, and documentation.
- **Human conversations in OpenCode remain in French**.

## Package Manager Policy

- Always use `pnpm` for installing, adding, and removing packages.
- Never use `npm` or `yarn`.

## Version Control

This project uses **Jujutsu** for version control.

**Never commit changes yourself.** Only the user may create commits. If you make changes, just write the files — do not stage or commit anything.
